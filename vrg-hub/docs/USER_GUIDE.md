## Reminders System

The Reminders feature helps you track important dates, compliance deadlines, and recurring tasks.

### Accessing Reminders

Navigate to **Reminders** from the sidebar menu.

### Key Features

#### Creating a Reminder
1. Click **"New Reminder"**
2. Fill in the details:
   - **Title** - Name of the reminder
   - **Description** - Additional details
   - **Due Date** - When the reminder is due
   - **Category** - Compliance, Maintenance, License, Training, etc.
   - **Priority** - Low, Medium, High, Urgent
   - **Recurrence** - One-time, Daily, Weekly, Monthly, Yearly
   - **Location/Brand** - Assign to specific location or brand

#### Notification Options
- **SMS Notifications** - Receive text message reminders
- **Email Notifications** - Get email alerts
- **In-App Notifications** - See alerts in the Hub

#### Notification Timing
Configure when you want to be notified:
- Days before due date (e.g., 7 days, 3 days, 1 day)
- On the due date
- After overdue (for follow-up reminders)

### Viewing Reminders

- **List View** - See all reminders in a sortable list
- **Calendar View** - Visual calendar with reminder dates
- **Dashboard** - Statistics and category breakdowns

### Managing Reminders

- **Mark Complete** - Click the checkbox or "Mark as Complete" button
- **Snooze** - Postpone reminder notifications
- **Archive** - Hide completed reminders from main view
- **Edit** - Modify any reminder details
- **Delete** - Permanently remove a reminder

### SMS Completion Links

When you receive an SMS reminder, it includes a link to quickly mark the reminder as complete from your mobile device.

## Daily Checklists

Daily checklists ensure consistent task completion across locations.

### Accessing Checklists

Navigate to **Daily Checklist** from the sidebar.

### Completing a Checklist

1. Select your location (if not auto-detected)
2. View the checklist items organized by time slot
3. For each item:
   - Click the checkbox to mark complete
   - Add initials if required
   - Add notes if needed
   - Mark as N/A if not applicable
4. Click **"Submit Checklist"** when finished

### Time Slots

Checklists are organized by time:
- **Opening** - Tasks at the start of day
- **Morning** - Mid-morning tasks
- **Midday** - Lunch period tasks
- **Afternoon** - Afternoon tasks
- **Closing** - End of day tasks

### Day-Specific Items

Some items only appear on certain days (e.g., weekly cleaning on Fridays).

## Print Order Forms

Order print materials for clinics and marketing purposes.

### Accessing Print Orders

Navigate to **Print Ordering** from the sidebar.

### Creating an Order

1. Click **"New Order"**
2. Select the brand and location
3. Choose items from the catalog:
   - Brochures
   - Business cards
   - Signage
   - Forms
   - Marketing materials
4. Specify quantities
5. Add delivery instructions
6. Submit for approval

### Order Status

Track your orders through these stages:
- **Draft** - Order being prepared
- **Pending Approval** - Awaiting manager approval
- **Approved** - Order confirmed
- **In Production** - Being printed
- **Shipped** - On the way
- **Delivered** - Order complete

## Request Management

Submit and track various types of requests.

### Request Types

- **Hardware Requests** - Equipment and hardware needs
- **Toner Requests** - Printer supplies
- **Marketing Requests** - Marketing materials and campaigns
- **User Account Requests** - New accounts and access changes
- **IT Service Desk** - Technical support
- **HR Requests** - Human resources matters
- **Facility Services** - Building and maintenance
- **Finance Requests** - Financial approvals

### Creating a Request

1. Click **"New Request"** or navigate to a specific request type
2. Fill in required fields:
   - Title and description
   - Brand and location
   - Priority level
   - Attachments (if needed)
3. Submit the request

### Tracking Requests

- **My Requests** - View your submitted requests
- **All Requests** - See all company requests (if permitted)
- **Approvals** - Pending items requiring your approval

### Request Workflow

1. **Draft** - Request being prepared
2. **Submitted** - Awaiting review
3. **In Progress** - Being worked on
4. **Pending Approval** - Needs manager sign-off
5. **Approved/Rejected** - Decision made
6. **Completed** - Request fulfilled

## User Roles & Permissions

### Role Types

- **Requester** - Basic user access
- **Manager** - Approval permissions
- **Tenant Admin** - Full company administration
- **Super Admin** - System-wide access
- **Content Editor** - Users with page/module editing permissions

### Content Editors

The following users have been granted permissions to create, edit, and manage custom pages and modules:

- **Steven Luu** - Full page/module editing access
- **Slav Kotevski** - Full page/module editing access

Content Editors can:
- Create new custom pages
- Edit existing pages and their content
- Add, remove, and reorder modules within pages
- Publish pages for all users to see
- Manage modalities and their configurations

### Permission Categories

| Category | Description |
|----------|-------------|
| `view_*` | View access to features |
| `create_*` | Create new items |
| `edit_*` | Modify existing items |
| `delete_*` | Remove items |
| `approve_*` | Approve submissions |
| `manage_*` | Full control over features |

### Feature-Specific Permissions

#### Reminders

- `clinic_setup.manage` - Full checklist management

#### Print Ordering
- `print_ordering.read` - View print orders

#### Roster Converter
